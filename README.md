# Проведение исследований алгоритмами машинного обучения
Выполнил: Ирисов Тамерлан М8О-409Б-22



Выбранные таргеты:

1) Классификация пола посетителей тренажерного зала по их признакам. Определение демографических характеристик клиентов позволяет персонализировать маркетинговые кампании, оптимизировать услуги и улучшать клиентский опыт. 
2) Определение цены умных часов по их техническим характеристикам. Выбор датасета обоснован практической потребностью в анализе ценообразования на рынке умных часов, что позволяет как производителям оптимизировать характеристики и цену новых моделей, так и покупателям принимать обоснованные решения при покупке.
